Design Principles Document

Therese Lee D. Gasmen

Proximity

Muji

muji.com/us

The Muji website is using the proximity principle to group items in the same category. It signifies a strong relationship between the elements that are close together so the viewer's eye can clearly fall into place without the need to read the labels. With this principle, the website creates a more appealing white space.

Alignment

Ikea

ikea.com

The Ikea website is using the alignment principle. No element falls on the page arbitrarily. You can clearly see the invisible line where the texts and the buttons fall and line up. It also follows only one alignment--the left alignment--which creates a visual connection with every information on the page. My eyes don't have to wander around to find important information I might have missed.

Repetition

Netflix

netflix.com

The Netflix website is using the repetition principle. With the way the films and shows are presented--a category on the top left and the sideways scrolling for recommended films--I can see the organization that unifies the web page and adds visual interest.
